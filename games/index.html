Overview

While Hyacinth is a bit unwieldly to put in many games, I have written some documentation on putting them in games. This page acts as a hub for different files, or a subwiki for documentation if necessary (for example, Dungeons and Dragons). I might also try coding some web-based games at somepoint.

Mod Pages

(Into The Breach) Hyacinthmod
- A custom player squad for Into the Breach, starring Hyacinth's puppets. Currently features a single squad with 3 mechs and 1 deployable, as well as fully custom weaponry.

Subwikies

(Dungeons and Dragons) Abomination Class
- (Subwiki) A relatively flexible/customizeable variant of Hyacinth's "species" and "class" made for Dungeons and Dragons 5th Edition. Occupies the role of a tanky frontline partcaster by default (similar to a Paladin), but can be elaborated into different roles via subclasses and other player choices.
